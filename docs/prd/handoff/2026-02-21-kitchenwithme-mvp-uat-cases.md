# 厨房陪我 MVP 产品验收用例（UAT）

## 1. 文档信息

- 版本：v0.1
- 日期：2026-02-21
- 适用范围：Web MVP
- 关联需求：`docs/prd/core/2026-02-21-kitchenwithme-mvp-prd.md`

## 2. 验收说明

- 优先级：P0 > P1 > P2
- 通过标准：
  - P0 全部通过
  - P1 关键用例通过
  - P2 至少主路径通过
- 文案与视觉需同时通过品牌一致性检查

## 4. 品牌一致性验收

| 检查项 | 通过标准 |
| --- | --- |
| 文案语气 | 无命令/催促/评判语句 |
| 禁忌词 | 不出现任务、打卡、KPI、管理 |
| 状态符号 | 不出现对勾、叉号、进度条 |
| 视觉风格 | 无纯黑主色、无高饱和硬对比 |

## 5. 关键指标验收

| 指标 | 目标值 |
| --- | --- |
| 主链路完成率 | > 60% |
| 语音响应延迟 P50 | < 1.2s |
| 语音响应延迟 P95 | < 2.0s |
| 中断响应 | < 400ms |
| 开火后记录提交率 | 持续提升（首版建立基线） |

## 6. 缺陷分级建议

- S1：主链路中断（无法完成做饭会话）
- S2：关键能力异常（打断/下一步/计时失效）
- S3：体验问题（文案不顺、状态反馈轻微错误）
- S4：视觉细节问题（间距、样式轻微偏差）
